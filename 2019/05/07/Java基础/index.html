标识符及其命名规则

  • 标识符：能够自己命名的
  • 命名规则：
    1. 只能由大小写字母，数字，下划线和$组成
    2. 开头不能是数字，关键字和保留字，但可以包含关键字和保留字
    3. Java中严格区分大小写
    4. 标识符不能包含空格

标识符命名规范

  • 包名：全部小写 xxyyzz
  • 类名，接口名：大驼峰规则 XxxYyyZzz
  • 方法名，变量名：小驼峰规则 xxxYyyZzz
  • 常量：全部大写 XXX_YYY_ZZZ
  • 注意：为提高可读性，尽量要“见名知意”
