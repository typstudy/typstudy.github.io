数组

发表于 2019-05-08 | 更新于 2019-05-12 | 分类于 学习

数组概述

  • 数组（Array）

    1. 多个相同类型数据按一定顺序排列的集合，并使用一个名字命名，并通过编号的方式对这些数据进行统一管理

    2. 数组本身就是引用数据类型，而数组的元素可以是任意数据类型，包括基本数据类型和引用数据类型

    3. 数组长度一旦确定，就不能修改

  • 分类

    1. 按维度分：一维数组，二维数组，….
    2. 按元素数据类型分：基本数据类型元素的数组，引用数据类型元素的数组

一维数组

  1. 声明方式：type[] var或type var[]
  2. 初始化：
    • 静态初始化：在定义数组的同时为数组元素分配空间并赋值。
    • 动态初始化：数组声明且为数组元素分配空间与赋值操作分开
  3. 数组元素引用：数组名[数组元素下标]，注意：下标从0开始
  4. 数组长度获取：数组名.length
  5. 数组元素默认初始化值：
    • 整型：0
    • boolean：false
    • 浮点型：0.0
    • char：0或‘\u0000’
    • 引用类型：null
  6. 遍历数组
  7. 内存解析

二维数组

  1. 概念：数组元素为一维数组
  2. 声明及初始化：
    • type[] [] var=new type[length] []
  3. 默认初始值：
    • 外层元素初始值：地址值
    • 内层元素初始值：跟一维数组一样
  4. 内存解析

数组常见算法

  1. 数组元素的赋值(杨辉三角，回行数等)
  2. 求数值型数组中元素的最大值，最小值，平均值，总和等
  3. 数组的复制，反转，查找(线性查找，二分法查找)
  4. 数组元素的排序(冒泡排序，快速排序，堆排序，归并排序)

数组工具类Arrays

  1. equals：判断两个数组是否相等
  2. toString：输出数组信息
  3. fill：将指定值填充到数组
  4. sort：对数组进行排序
  5. binarySearch：对数组进行二分法查找指定值
